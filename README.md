# PROYECTO-DE-CLASIFICACION-DE-SENALES-DE-AUDIO-CON-RNN-Y-CNN
Sistema de clasificación automática de género por voz implementado con redes neuronales profundas. Se comparan dos enfoques: una Red Neuronal Convolucional (CNN) basada en mel-espectrogramas y una Red Neuronal Recurrente (RNN-LSTM) basada en coeficientes MFCC, aplicados sobre el dataset Gender Recognition by Voice de Kaggle.
